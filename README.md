# 591 租屋列表過濾 Chrome 擴充套件

## 目標
- 在 `https://rent.591.com.tw/list` 網頁上，使用者可以「移除」特定房屋物件，
 使之即時不顯示，且重新載入或下次造訪時 *不再* 顯示相同物件。

## 功能需求
1. 在列表頁載入時，讀取已標記移除的物件清單並隱藏對應元素。
2. 在每個房屋物件區塊新增「移除」按鈕，點擊後立即隱藏該區塊，並將物件 ID 儲存至本地紀錄。
3. 提供選項頁面（Options Page），可檢視/還原已移除的物件清單。

## 技術選型
- **Manifest v3**：使用 Chrome 擴充套件新規範
- **Content Script**：解析並操作 `rent.591.com.tw/list` 頁面 DOM
- **Chrome Storage API**：`chrome.storage.local` 作為本地儲存
- **Options Page**：管理已移除清單，提供匯入/匯出功能

## 功能說明

### 1. 內容腳本功能
- 在591租屋網站列表頁自動添加「移除」按鈕到每個房屋物件
- 點擊按鈕後立即隱藏該物件並將其ID儲存到本地
- 頁面載入時自動隱藏先前移除過的物件
- 添加浮動設定按鈕，可快速開啟選項頁面

### 2. 選項頁面功能
- 顯示所有已移除物件清單
- 提供搜尋功能，可快速找到特定物件
- 支援還原/刪除單個物件
- 支援清除所有物件
- 提供資料匯出/匯入功能，便於備份或轉移設定

## 使用方式

### 安裝擴充套件
1. 下載此專案的所有文件
2. 開啟Chrome瀏覽器，進入擴充功能頁面 (chrome://extensions/)
3. 開啟右上角的「開發人員模式」
4. 點擊「載入未封裝項目」，選擇包含這些文件的資料夾
5. 完成安裝後，瀏覽器右上角會出現擴充套件圖示

### 使用擴充功能
1. 前往 [591租屋網](https://rent.591.com.tw/list)
2. 瀏覽房屋列表時，每個物件右上角會出現「✕ 移除」按鈕
3. 點擊按鈕可立即隱藏不想看的物件，下次瀏覽時也不會顯示
4. 使用右下角的⚙️按鈕可開啟管理介面
5. 在管理介面中可查看、搜尋、還原或匯出已移除的物件清單

## 功能特色

### 智能物件識別
- 多重檢測方式自動識別頁面中的房屋物件
- 即使591網站更新頁面結構，仍能保持功能正常

### 應急模式
- 如無法正常添加移除按鈕時，自動啟用浮動控制面板
- 提供物件標記功能，方便用戶識別和移除物件

### 完整資料管理
- 支援匯出已移除物件列表為JSON文件
- 可匯入先前的設定或在不同裝置間同步
