# 隱私權政策 / Privacy Policy

**591 租屋列表過濾** Chrome 擴充套件

最後更新：2026 年 2 月

---

## 中文版

### 資料收集

本擴充套件 **不會收集、傳輸或分享任何個人資料**。

### 資料儲存

本擴充套件僅使用 Chrome 的本機儲存空間（`chrome.storage.local`）來儲存以下資料：

- 您選擇移除的 591 租屋物件 ID
- 移除時間戳記
- 物件名稱（用於顯示）

所有資料 **完全儲存在您的瀏覽器中**，不會傳送到任何外部伺服器。

### 資料控制

您可以隨時透過擴充套件的選項頁面：

- 查看已儲存的資料
- 還原個別物件
- 清除所有資料
- 匯出或匯入資料

### 權限說明

| 權限 | 用途 |
|------|------|
| `storage` | 儲存已移除的物件清單於本機 |
| `host_permissions` (rent.591.com.tw) | 在 591 租屋列表頁面注入移除按鈕 |

### 第三方服務

本擴充套件 **不使用任何第三方分析、追蹤或廣告服務**。

### 聯絡方式

如有任何隱私相關問題，請透過 Chrome Web Store 的支援頁面聯繫。

## English Version

### Data Collection

This extension **does not collect, transmit, or share any personal data**.

### Data Storage

This extension only uses Chrome's local storage (`chrome.storage.local`) to store:

- IDs of 591 rental listings you choose to remove
- Removal timestamps
- Listing names (for display purposes)

All data is stored **entirely in your browser** and is never sent to any external server.

### Data Control

You can at any time through the extension's options page:

- View stored data
- Restore individual items
- Clear all data
- Export or import data

### Permissions

| Permission | Purpose |
|------------|---------|
| `storage` | Store removed items list locally |
| `host_permissions` (rent.591.com.tw) | Inject remove buttons on 591 rental listing pages |

### Third-Party Services

This extension **does not use any third-party analytics, tracking, or advertising services**.

### Contact

For any privacy-related questions, please contact through the Chrome Web Store support page.
